# PDF Viewer SDK 

## Description

This project is a JavaScript-based application with examples for plain HTML usage. It includes an `index.esm.js` module in the `src` directory and a Rollup configuration for bundling.

## Project Structure

```
package.json         # Project dependencies and scripts
rollup.config.js     # Rollup configuration for bundling
examples/            # Example usage of the project
  plain-html/        # Plain HTML example
    index.html       # HTML file for testing
    samples/         # Additional sample files
src/                 # Source code
  index.esm.js       # Main module entry point
```

## Getting Started

### Install Dependencies

Run the following command to install dependencies:

```sh
npm install
```

### Build the Project

To build the project using Rollup, run:

```sh
npm run build
```

### Run Examples

Open the `examples/plain-html/index.html` file in your browser to see the project in action.
